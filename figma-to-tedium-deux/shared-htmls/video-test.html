How to add more videos:

  1. Place video files in the shared-htmls/video/ directory
  2. Supported formats: MP4, WebM, OGG, AVI, MOV, WMV, FLV
  3. Access videos at: http://localhost:3000/shared-htmls/video/filename.mp4
  4. Use in HTML: <video src="video/filename.mp4"></video>
